# hits

## Getting started
```bash
# Copy all .env.example files into your .env file

# Install dependencies
$ yarn

# Run Docker Compose 
$ docker-compose up -d`

$ cd api/prisma && go run github.com/prisma/prisma-client-go db push

# Start Go API
$ cd api && go run main.go
```
